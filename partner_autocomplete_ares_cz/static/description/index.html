Partner Autocomplete Ares CZ

Completes Partner information using ARES from https://wwwinfo.mfcr.cz/

Availiable countries
  1. Czech Republic
Information that can be completed:
  1. Address
  2. Tax ID
  3. Dynamically mappable fields (see Configuration point 4.):
    • Company identification number

Configuration

  1. Go to Settings > General Settings > Contacts
  2. Enable Partner Autocomplete
  3. Choose provider ARES.cz
  4. Choose fields for mapping (optional)

Dynamically mappable fields used to paste data from ARES.cz to any field in res.partner(Contact) model.

Autocomplete During Import

  1. Create import file with columns: name and import_enrich_company (it can have other columns as well)
  2. In the import_enrich_company column, enter CIN
  3. Import the file

Manual Update of Records

  1. Go to Contacts
  2. Switch to the list view
  3. Select records you want to update
  4. Click Action > Update
